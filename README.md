# Games explorer
Games explorer app was created as a research project with no practical purpose in mind.<br>You can browse game genres and stores. Also, you can browse games which belong same genre and read information about selected game. Every store leads you to the official web site. <br><br>
Made with <b>SwiftUI</b> and <b>Combine </b>. Targets iOS 15.0 or higher.<br>
Feel free to use, modify, redistribute it without any restrictions. <br>
